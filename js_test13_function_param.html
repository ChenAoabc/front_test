什么是函数显式参数于隐式参数 ？

显式参数: 在函数定义时列出的参数，就是显式参数。 例如：function myFunction(param1, param2)。

隐式参数: 在函数调用时传递给函数真正的值，就是隐式参数。 例如：function（1，2）

Note: 1、js函数定义显式参数时，没有指定数据类型。 2、js对隐式参数没有进行类型检查，可以传递任意类型的数据。 3、js函数对隐式参数的个数没有进行检查，可以传递任意数量的参数。

默认参数:

在ES5中如果函数在调用时，未提供相应的隐式参数，则参数的默认值为：undefined

在ES6中，可以使用默认参数来指定参数的默认值，语法如下：

function myFunction(param1, param2 = 10) {}

在调用函数时，如果未提供参数，则使用默认值。

注意：

1、默认参数只适用于最后一个参数，前面的参数不能有默认值。

2、默认参数的值是只读的，不能被修改。

3、默认参数的值在函数定义时计算一次，而不是在函数每次调用时计算。

函数参数的传递方式：

1、值传递：函数调用时，将实参的值复制给形参，实参的值发生变化，不会影响到形参的值。

2、引用传递：函数调用时，将实参的地址复制给形参，实参的值发生变化，会影响到形参的值。

3、可变参数：函数调用时，将实参的元素复制给形参，实参的值发生变化，会影响到形参的值。

4、不定长参数：函数调用时，将实参的元素复制给形参，实参的值发生变化，会影响到形参的值。

函数参数的作用域：

1、函数内部声明的变量，只在函数内部有效，外部不可访问。

2、函数内部的变量，可以被重新赋值，外部也可以访问。

3、函数的参数在函数调用时，会被复制给形参，形参的值不会影响实参的值。

函数参数的注意事项：

1、函数参数的数量和类型必须与函数定义时一致。

2、函数参数的顺序必须与函数定义时一致。

3、函数参数的名称只在函数内部有效，外部不可访问。

4、函数参数的作用域只在函数内部有效，外部不可访问。

5、函数参数的默认值只在函数定义时有效，函数调用时不可修改。

6、函数参数的传递方式有值传递、引用传递、可变参数、不定长参数。

arguments 对象

arguments 对象是一个类数组对象，包含了函数调用时的所有参数， 可以通过索引来访问。

note: arguments 对象不是一个真正的数组，它只是一个类似数组的对象，

它的索引属性值从 0 开始，到 length-1 结束。

arguments 对象有 length 属性，表示它包含的参数个数。

arguments 对象有 callee 属性，表示它所属的函数。
