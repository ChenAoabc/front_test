JavaScript Test 30: BOM Window cookie API

什么是Cookie？

cookie，也称为小型文本文件，存储在用户的浏览器上， 用于存储网站的用户信息。 例如，当用户退出访问当前页面后，就会与服务器断开连接， 而用户的身份信息可以保存在cookie中 当下次用户访问该页面时，浏览器会自动发送cookie给服务器， 服务器根据cookie内容就可以验证用户信息，不需要用户再次登录。 cookie已下面的形式存储数据（键值对） name=value;

如何使用js创建cookie？

可以使用document.cookie属性来创建、读取、删除cookie。 语法： document.cookie = "name=value; expires=expiration_date; path=path; domain=domain"; 其中，name和value是必需的，其他属性可选。
