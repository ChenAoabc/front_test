什么是BOM（Browser Object Model）？

BOM（Browser Object Model）是指浏览器对象模型，它提供了独立于内容而与浏览器窗口 进行交互的对象。BOM定义了与浏览器窗口进行交互的对象，如window、navigator、screen、 history、location等。BOM的主要作用是为开发者提供与浏览器窗口进行交互的能力， 如操控浏览器窗口、获取用户信息、操作cookie、使用本地存储等。

BOM的常用属性和方法:

  • window：返回当前窗口的窗口对象。
  • navigator：返回包含有关浏览器的信息的navigator对象。
  • screen：返回包含有关屏幕的信息的screen对象。
  • history：返回包含有关历史记录的信息的history对象。
  • location：返回包含有关当前页面的URL信息的location对象。
  • document：返回当前文档的document对象。
  • setTimeout()：设置一个定时器，在指定的时间后执行一个函数。
  • clearTimeout()：取消一个setTimeout()设置的定时器。
  • setInterval()：设置一个定时器，每隔指定的时间就执行一个函数。
  • clearInterval()：取消一个setInterval()设置的定时器。
