BOM Window 设置定时器

定时器可以用来执行某些操作，比如每隔一段时间执行某些代码， 或者在指定的时间执行某些代码。

在 JavaScript 中，可以使用 setTimeout() 方法来设置定时器， 该方法的作用：在指定时间后执行代码 该方法接收两个参数：要执行的代码和执行的延迟时间（以毫秒为单位）。 该方法返回一个 ID，可以用来取消定时器。

使用 clearTimeout() 方法可以取消定时器。

在 JavaScript 中，可以使用 setInterval() 方法来设置定时器， 该方法的作用：在指定时间里循环执行代码 该方法接收两个参数：要执行的代码和执行的延迟时间（以毫秒为单位）。 该方法返回一个 ID，可以用来取消定时器。
