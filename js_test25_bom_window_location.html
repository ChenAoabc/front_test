JavaScript Test 25: BOM Window Location Object

window.location 对象用以获得当前页面的地址（URL） 并将浏览器重新定向到新的页面。

window.location.href 获得当前页面的完整 URL。

window.location.protocol 获得当前页面的协议。

window.location.host 获得当前页面的域名主机。

window.location.hostname 获得当前页面的域名。

window.location.port 获得当前页面的端口号。

window.location.pathname 获得当前页面的路径名。

window.location.search 获得当前页面的查询字符串。

window.location.hash 获得当前页面的锚点。

window.location.assign() 方法可以重新定向到新的页面。

window.location.replace() 方法可以将当前页面替换成新的页面。

window.location.reload() 方法可以重新加载当前页面。
